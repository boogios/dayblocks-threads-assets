# DayBlocks Threads Assets

Public, English-language media assets used by the official DayBlocks Threads publishing workflow.

Public base URL: `https://boogios.github.io/dayblocks-threads-assets`

## Files

- `dayblocks-plan.png`
- `dayblocks-see.png`
- `dayblocks-recap-threads.mp4`
- `dayblocks-widgets.jpg`
- `dayblocks-app-store-card.jpg`

These assets contain no credentials, user data, or application source code.

The images use the verified `en-US` DayBlocks UI. The recap video is an 18-second H.264/AAC sequence of the English Plan, Do, See, and Widget screens.
